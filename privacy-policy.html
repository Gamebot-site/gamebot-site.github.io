Privacy Policy


View

What Data is stored?

The following data may be collected and stored when users uses a command. No data is collected automatically.
- User ids
- Server ids
- Text Channel ids

Why is this data required?

We need this information to provide the basic services we offer to you. It includes configurations set per server regarding bot behaviour in different channel and other personalizations.

How do I get my data removed?

You can remove all your data using the configuration commands. If you have any concerns, you can contact us using g.staff
